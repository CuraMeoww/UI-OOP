# 🎬 GIAO DIỆN BÁN HÀNG RẠP CHIẾU PHIM

---

## 🚀 1. Giới thiệu demo
Demo được xây dựng nhằm mô phỏng trải nghiệm thực tế tại một rạp chiếu phim thông minh. Hệ thống cho phép người dùng khám phá các bộ phim mới nhất, thực hiện quy trình đặt vé, chọn ghế và mua đồ ăn kèm một cách trực quan.

## 🌟 2. Các tính năng chính

### 🎫 Phía Khách hàng (User App)
* **Trang chủ Cinematic:** Hiển thị danh sách phim với hiệu ứng Hover mượt mà.
* **Lịch chiếu thông minh:** Tự động kiểm tra logic ngày công chiếu để đóng/mở suất chiếu.
* **Đặt vé & Chọn ghế:** Giao diện chọn ghế thực tế, hỗ trợ đa lựa chọn.
* **Dịch vụ ăn uống:** Menu tích hợp Combo tiết kiệm, tự động tính tiền.

### 📊 Phía Quản trị (Admin Panel)
* **Dashboard tối giản:** Tập trung vào 3 mục chính: Phim, Đồ ăn và Thống kê.
* **Quản lý Kho thực tế:** Cập nhật số lượng tồn kho ngay khi phát sinh giao dịch.
* **Thống kê doanh thu:** Tự động tính toán Doanh thu, Tổng vé và Phim được yêu thích nhất.

## 🛠️ 3. Công nghệ sử dụng (Tech Stack)
* **Ngôn ngữ:** Java (JDK 17+)
* **Giao diện:** Java Swing & AWT.
* **Lưu trữ:** File System (`.txt`).

## 📦 4. Hướng dẫn cài đặt (Setup & Run)
1.  **Clone:** Tải mã nguồn về máy.
2.  **Import:** Mở bằng IntelliJ IDEA.
3.  **Data:** Kiểm tra các file `foods.txt`, `accounts.txt` nằm ở thư mục gốc.
4.  **Run:** Thực thi `Main.java` để bắt đầu.

## 📂 5. Cấu trúc thư mục (Project Structure)
* `src/model`: Các lớp đối tượng dữ liệu.
* `src/view`: Giao diện các màn hình chính.
* `src/util`: File `Config.java` chứa bảng màu và font chữ hệ thống.
* `src/assets`: Kho chứa Poster phim và icon.
